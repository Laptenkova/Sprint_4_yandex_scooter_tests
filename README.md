<h2>Финальный проект 4 спринта</h2>

<h3>Описание проекта</h3>
Проект содержит автотесты для учебного сервиса "Яндекс Самокат". 
Реализованы тесты:
* проверка выпадающего списка "Вопросы о важном"
* оформление заказа через разные кнопки (верхнюю и нижнюю)

<h3>Технологии</h3>
* Язык программирования: Java 11
* Фреймворк для тестирования: JUnit 4.13.2
* Библиотека для автоматизации браузера: Selenium WebDriver 4.34.0
* Сборка проекта: Maven 3.9.0
* Поддерживаемые браузеры: Google Chrome (Версия: 139.0.7258.67 ), Firefox (Версия: 141.0.3)


<h3>Структура проекта</h3>
`src/main/java` — основной код проекта<br>
`src/test/java` — тесты<br>
Пакет page — содержит Page Object классы для страниц приложения.<br>
Настройки сгруппированы в отдельные утилитные классы. <br>
Тесты организованы по функциональным классам с использованием параметризации.

<h3>Настройка проекта</h3>
* Установите Java 11
* Установите Maven последней версии (3.9.0)
* Установите браузеры Google Chrome и Mozilla Firefox последних версий

<h3>Запуск тестов</h3>
Выполните в командной строке: mvn clean test

<h3>Особенности реализации</h3>
1. Page Object Pattern
2. Параметризованные тесты (2 набора данных)
3. Поддержка Chrome и Firefox
4. Централизованное управление WebDriver

<h3>Известные проблемы</h3>
Баг в Chrome при оформлении заказа (ожидаемое падение теста)
